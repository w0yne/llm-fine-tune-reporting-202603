# LLM Translation Fine-Tuning — CDP Project Update

Slidev presentation for a CDP project update on LLM translation model fine-tuning.

## Overview

This deck covers:
- **Phase 1 Review**: Qwen 8B fine-tuning PoC results and business feedback
- **Phase 2 Results**: Teacher-Student architecture with adversarial prompt optimization
- **Next Steps**: Phased plan from validation to production

## Tech Stack

- [Slidev](https://sli.dev/) — Presentation framework
- [apple-basic](https://github.com/slidevjs/themes/tree/main/packages/theme-apple-basic) theme
- Vue components from [consulting-slides-demo](https://github.com/w0yne/consulting-slides-demo)

## Usage

```bash
npm install
npm run dev      # Dev server
npm run build    # Build for deployment
npm run export   # Export to PDF
```
